Overview
********

``paintbox`` has the followong requirements:

- `Python <https://www.python.org/>`_ 3.7 or later
- `astropy <https://www.astropy.org/>`_ 4.0 or later

Optional requirements for tutorials

- pymc3
- ppxf
- matplotlib
